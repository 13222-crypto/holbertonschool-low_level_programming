# C - File I/O

## Description
This project covers the basics of file handling in C using system calls:
* How to create, open, close, read, and write files.
* Understanding file descriptors and permissions.
* Using system calls like `open`, `read`, `write`, and `close`.

## Files
| File | Description |
| --- | --- |
| 0-read_textfile.c | Function that reads a text file and prints it to the POSIX standard output. |

## Requirements
* Compiled on Ubuntu 20.04 LTS.
* Uses the Betty style.
