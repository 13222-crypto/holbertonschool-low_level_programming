# Doubly Linked Lists

## Description
This project covers the implementation and usage of doubly linked lists in C.

## Requirements
* All files compiled on Ubuntu 20.04 LTS using `gcc`.
* Code follows the Betty style.
* Allowed functions: `malloc`, `free`, `printf`, and `exit`.

## Files
| File | Description |
| --- | --- |
| 0-print_dlistint.c | Function that prints all elements of a dlistint_t list. |
