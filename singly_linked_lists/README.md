# Singly Linked Lists

## Description
This project focuses on understanding and implementing singly linked lists in C. It covers basic operations such as printing nodes, counting elements, and managing memory within a linked list structure.

## Learning Objectives
* Explain when and why to use linked lists versus arrays.
* Build and use linked lists effectively.

## Requirements
* **Environment:** Ubuntu 20.04 LTS
* **Compiler:** gcc (options: -Wall -Werror -Wextra -pedantic -std=gnu89)
* **Style:** Betty style compliant.
